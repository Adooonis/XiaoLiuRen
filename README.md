# 小六壬神课 🧙‍♂️

## 📱 安装到手机桌面

### Android 手机
1. 用手机浏览器打开网页
2. 点击浏览器菜单中的"添加到主屏幕"
3. 确认添加，应用图标将出现在桌面
4. 点击桌面图标即可独立使用

### iPhone 手机
1. 用Safari浏览器打开网页
2. 点击Safari底部分享按钮
3. 向下滑动找到"添加到主屏幕"
4. 点击"添加"，应用将出现在主屏幕
5. 点击桌面图标即可独立使用

## ✨ 功能特点

### 🎯 核心功能
- **传统小六壬算法**：遵循古法大安→留连→速喜→赤口→小吉→空亡
- **双重起卦方式**：随机数字或阴历日期时辰
- **详细卦象解读**：每个神祇的详细解释和含义
- **完整占卜流程**：从起卦到结果，完整传承

### 📲 移动端优化

- **PWA应用**：可安装到桌面，像原生应用一样使用
- **离线功能**：无需网络也可正常使用
- **触觉反馈**：移动设备震动反馈增强体验

### 🎨 传统美学

## 🔮 使用方法

### 1. 输入问题
在"请输入您想测算的问题"文本框中输入您想占卜的具体问题，如：
- "今日出行是否顺利？"
- "本月工作运势如何？"
- "感情发展怎样？"

### 2. 选择起卦方式
- **随机三数**：手动输入三个1-999之间的数字
- **阴历时辰**：选择阴历月、日、时辰

### 3. 开始测算
点击"开始测算"按钮，应用将按照传统小六壬算法进行推演

### 4. 查看结果
测算完成后，您将看到：
- **卦象总览**：三个用神按顺序显示
- **六神详解**：每个神祇的详细解释
- **生成AI指令**：AI指令格式的询问内容

## 🎨 界面说明

### 颜色含义
- 🟢 **绿色**：吉利神祇（大安、速喜、小吉）

## 🛠️ 故障排除

### 安装问题
- **无法安装PWA**：确认使用Chrome、Safari等现代浏览器
- **图标不显示**：清除浏览器缓存后重试
- **添加失败**：检查浏览器版本是否为最新

### 显示问题
- **文字显示异常**：确认设备支持中文显示
- **布局错乱**：刷新页面或更换浏览器
- **颜色不正确**：确认设备支持CSS颜色

### 功能问题
- **输入无响应**：检查输入框是否聚焦，尝试重新点击
- **计算错误**：确认输入完整的数字和选择
- **页面空白**：刷新页面或重启浏览器
