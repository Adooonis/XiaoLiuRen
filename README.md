# 小六壬神课 🧙‍♂️

## 📱 应用介绍

小六壬神课是一款基于传统六壬起卦算法的占卜应用。遵循古法大安→留连→速喜→赤口→小吉→空亡的循环计数，为用户提供纯传统的起卦测算服务。无需网络连接，完全基于本地计算，保护您的隐私。

## 📲 安装到手机桌面

### Android 手机
1. 用手机浏览器打开网页
2. 点击浏览器菜单中的"添加到主屏幕"
3. 确认添加，应用图标将出现在桌面
4. 点击桌面图标即可独立使用

### iPhone 手机
1. 用Safari浏览器打开网页
2. 点击Safari底部分享按钮
3. 向下滑动找到"添加到主屏幕"
4. 点击"添加"，应用将出现在主屏幕
5. 点击桌面图标即可独立使用

## 🔮 使用方法

### 1. 输入问题
在"请输入您想测算的问题"文本框中输入您想占卜的具体问题，如：
- "今日出行是否顺利？"
- "本月工作运势如何？"
- "感情发展怎样？"
- "投资项目可否进行？"

### 2. 选择起卦方式
#### 方式一：随机三数
- 手动输入三个1-999之间的数字
- 可以是任意有意义的数字组合

#### 方式二：阴历时辰
- 选择阴历月份（1-12月）
- 选择阴历日期（1-30日）
- 选择时辰（子时、丑时、寅时等十二时辰）

### 3. 开始测算
点击"开始测算"按钮，应用将按照传统小六壬算法进行推演

### 4. 查看结果
测算完成后，您将看到：
- **卦象总览**：三个用神按顺序显示，包含测算时间、起卦数字、测算问题
- **六神详解**：每个神祇的详细解释和传统含义
- **生成AI指令**：标准化的AI询问指令模板
- **一键复制**：点击"复制指令"按钮即可将完整AI指令复制到剪贴板

## 🤖 AI指令生成功能

### AI指令格式说明
应用会生成标准化的AI询问指令，格式如下：
```
你是一个专业的小六壬卦象分析师。用户通过小六壬起卦，得到的卦象是：[神祇1] → [神祇2] → [神祇3]。

测算问题：[您输入的问题]
起卦数字：[数字1]、[数字2]、[数字3]
当前时间：[当前时间]

请从以下几个方面进行详细分析：
1. 卦象含义：解释每个神兆的吉凶寓意
2. 整体运势：分析当前事情的整体发展趋势
3. 时机分析：判断做事的有利时机
4. 注意事项：指出需要防范的问题
5. 具体建议：给出切实可行的行动建议

请用温暖、专业、易懂的语言进行分析，字数控制在800-1200字左右。
```

### 使用方法
1. 完成测算后，找到"生成AI指令"区域
2. 查看生成的标准化AI指令内容
3. 点击"📋 复制指令"按钮
4. 将复制的指令粘贴到ChatGPT、Claude、DeepSeek等AI工具中
5. 获得更详细的占卜分析和建议
