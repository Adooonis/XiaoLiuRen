# 小六壬神课 🧙‍♂️

## 📱 应用介绍

小六壬神课是一款基于传统六壬起卦算法的占卜应用。遵循古法大安→留连→速喜→赤口→小吉→空亡的循环计数，为用户提供传统的起卦测算服务。现在还集成了AI深度分析功能，基于阿里云通义千问API为用户提供智能卦象分析。

## 📲 安装到手机桌面

### Android 手机
1. 用手机浏览器打开网页
2. 点击浏览器菜单中的"添加到主屏幕"
3. 确认添加，应用图标将出现在桌面
4. 点击桌面图标即可独立使用

### iPhone 手机
1. 用Safari浏览器打开网页
2. 点击Safari底部分享按钮
3. 向下滑动找到"添加到主屏幕"
4. 点击"添加"，应用将出现在主屏幕
5. 点击桌面图标即可独立使用

## ✨ 功能特点

### 🎯 核心功能
- **传统小六壬算法**：遵循古法大安→留连→速喜→赤口→小吉→空亡
- **双重起卦方式**：随机数字或阴历日期时辰
- **详细卦象解读**：每个神祇的详细解释和含义
- **完整占卜流程**：从起卦到结果，完整传承
- **AI深度分析**：基于阿里云通义千问API的智能卦象分析
- **AI指令生成**：生成标准化AI指令，可用于各种AI工具进行详细分析

### 🤖 AI分析功能
- **实时AI分析**：自动调用通义千问API进行深度分析
- **多维度解读**：从卦象含义、整体运势、时机分析、注意事项、具体建议五个方面分析

- **详细报告**：800-1200字的深度分析报告
- **智能建议**：根据卦象给出具体可行的行动建议

- **离线功能**：无需网络也可正常使用（基础测算功能）
- **触觉反馈**：移动设备震动反馈增强体验
- **响应式设计**：完美适配各种屏幕尺寸

## 🔮 使用方法

### 1. 输入问题
在"请输入您想测算的问题"文本框中输入您想占卜的具体问题，如：
- "今日出行是否顺利？"
- "本月工作运势如何？"
- "感情发展怎样？"

### 2. 选择起卦方式
- **随机三数**：手动输入三个1-999之间的数字
- **阴历时辰**：选择阴历月、日、时辰

### 3. 开始测算
点击"开始测算"按钮，应用将按照传统小六壬算法进行推演

### 4. 查看结果
测算完成后，您将看到：
- **卦象总览**：三个用神按顺序显示，包含测算时间、起卦数字、测算问题
- **六神详解**：每个神祇的详细解释和含义
- **生成AI指令**：标准化的AI询问指令
- **AI深度分析**：基于通义千问API的智能分析报告
- **一键复制**：点击复制按钮即可将AI指令复制到剪贴板

## 🤖 AI功能使用

### 自动AI分析
- 测算完成后，系统会自动调用阿里云通义千问API
- 进行多维度的专业卦象分析
- 生成800-1200字的详细分析报告
- 包含时机判断、注意事项、具体建议等

### AI指令生成
应用会生成标准化的AI询问指令，格式如下：
```
你是一个专业的小六壬卦象分析师。用户通过小六壬起卦，得到的卦象是：[神祇1] → [神祇2] → [神祇3]。

测算问题：[您输入的问题]
起卦数字：[数字1]、[数字2]、[数字3]
当前时间：[当前时间]

请从以下几个方面进行详细分析：
1. 卦象含义：解释每个神兆的吉凶寓意
2. 整体运势：分析当前事情的整体发展趋势
3. 时机分析：判断做事的有利时机
4. 注意事项：指出需要防范的问题
5. 具体建议：给出切实可行的行动建议

请用温暖、专业、易懂的语言进行分析，字数控制在800-1200字左右。
```

### 一键复制功能
- 点击"复制指令"按钮即可将完整指令复制到剪贴板
- 支持现代Clipboard API和传统复制方法
- 适用于移动端和桌面端

### 在其他AI工具中使用
将复制的指令粘贴到ChatGPT、Claude、DeepSeek、智谱清言等AI工具中，获得更详细的占卜分析和建议。

## 🛠️ 故障排除

### 安装问题
- **无法安装PWA**：确认使用Chrome、Safari等现代浏览器
- **图标不显示**：清除浏览器缓存后重试
- **添加失败**：检查浏览器版本是否为最新

### 显示问题
- **文字显示异常**：确认设备支持中文显示
- **布局错乱**：刷新页面或更换浏览器
- **颜色不正确**：确认设备支持CSS颜色

### 功能问题
- **输入无响应**：检查输入框是否聚焦，尝试重新点击
- **计算错误**：确认输入完整的数字和选择
- **复制失败**：检查浏览器权限，确认剪贴板访问权限
- **页面空白**：刷新页面或重启浏览器

### AI分析问题
- **AI分析失败**：检查网络连接，API服务可能暂时不可用
- **分析结果异常**：确认起卦数字和问题的正确性
- **网络超时**：AI分析需要较长时间，请耐心等待
- **API错误**：显示手动配置方法提示
