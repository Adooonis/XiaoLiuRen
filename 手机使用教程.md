## 📲 第一步：安装应用到手机

### Android 手机安装教程

#### 方法一：Chrome浏览器安装（推荐）
1. **打开Chrome浏览器**
   - 确保Chrome是最新版本
   - 在地址栏输入应用网址

2. **安装到桌面**
   - 点击Chrome右上角的三点菜单（⋮）
   - 选择"添加到主屏幕"
   - 或者点击底部"安装"按钮

3. **确认安装**
   - 在弹出的确认对话框中点击"安装"
   - 等待下载和安装完成

4. **使用应用**
   - 安装完成后，桌面会出现"小六壬"应用图标
   - 点击图标即可像原生应用一样使用

#### 方法二：其他浏览器安装
- **Edge浏览器**：菜单 → "安装此网站"
- **Firefox浏览器**：菜单 → "添加到主屏幕"
- **夸克浏览器**：菜单 → "添加桌面快捷方式"

### iPhone 手机安装教程

#### Safari浏览器安装（唯一方法）
1. **打开Safari浏览器**
   - 确保iOS版本在13以上
   - 打开应用网址

2. **添加到主屏幕**
   - 点击Safari底部中央的分享按钮
   - 向下滚动找到"添加到主屏幕"
   - 点击进入

3. **自定义名称**
   - 可以修改应用名称（默认为"小六壬"）
   - 点击右上角"添加"

4. **使用应用**
   - 应用图标会出现在主屏幕
   - 点击图标即可独立使用

## 🔮 第三步：学会起卦方法

### 方法一：随机三数起卦
这是最常用的起卦方式，适合初学者：

1. **选择"随机三数"**
   - 默认已选中
   - 界面显示三个数字输入框

2. **输入三个数字**
   - 每个数字范围：1-999
   - 可以输入任意数字
   - 建议：
     - 想到问题时随意想三个数字
     - 或者用当前的日期时间组合
     - 保持内心的专注和诚意

3. **示例**
   ```
   问题：今天出差顺利吗？
   数字：125, 367, 289
   ```

### 方法二：阴历时辰起卦
这是传统起卦方式，更具古法意味：

1. **选择"阴历时辰"**
   - 切换到传统起卦方式

2. **选择阴历月份**
   - 正月到腊月（1-12月）

3. **选择阴历日期**
   - 1日到30日

4. **选择时辰**
   ```
   子时 (23:00-1:00)   丑时 (1:00-3:00)
   寅时 (3:00-5:00)    卯时 (5:00-7:00)
   辰时 (7:00-9:00)    巳时 (9:00-11:00)
   午时 (11:00-13:00)  未时 (13:00-15:00)
   申时 (15:00-17:00)  酉时 (17:00-19:00)
   戌时 (19:00-21:00)  亥时 (21:00-23:00)
   ```

## 📖 第四步：问题输入技巧

### 什么样的问题效果好？

#### ✅ 推荐的问题类型
- **具体明确**："本月15号的会议能否成功？"
- **时间范围明确**："这周的工作安排是否顺利？"
- **单一事项**："这次跳槽机会是否适合我？"
- **情感问题**："我和TA的关系发展方向如何？"

#### ❌ 避免的问题类型
- **过于宽泛**："我的命运如何？"（太笼统）
- **多个问题**："工作和感情哪个更好？"（涉及多事项）
- **过于复杂**："我的人生规划该怎么做？"（需要详细分析）
- **时间不明**："我什么时候能结婚？"（没有时间范围）

### 问题示例对照

| ❌ 不好问题 | ✅ 好问题 |
|------------|----------|
| 我今年运气怎么样？ | 我这个月的财运如何？ |
| 我的感情生活好吗？ | 我和男朋友的关系发展如何？ |
| 投资股票赚钱吗？ | 我现在买这只股票合适吗？ |
| 工作好不好？ | 我换工作这个决定正确吗？ |

### 解读要点

#### 三个用神的意思
- **第一个神**：起始状态，代表事情的开端
- **第二个神**：发展过程，代表中间的变化
- **第三个神**：最终结果，代表事情的发展方向

#### 神祇颜色含义
- 🟢 **绿色（吉利）**：大安、速喜、小吉
- 🔴 **红色（谨慎）**：留连、赤口、空亡

#### 六神特性速查
| 神祇 | 特性 | 建议 |
|------|------|------|
| 🟢 大安 | 稳健、平安 | 静待时机，稳步前进 |
| 🔴 留连 | 缓慢、反复 | 耐心等待，不宜急躁 |
| 🟢 速喜 | 快速、吉利 | 把握机会，积极行动 |
| 🔴 赤口 | 纠纷、谨慎 | 谨言慎行，避免冲突 |
| 🟢 小吉 | 平稳、收获 | 稳步推进，小有收获 |
| 🔴 空亡 | 未定、模糊 | 观察等待，时机未到 |

## 🤖 第六步：使用AI功能

### 复制AI指令
1. **查看指令内容**
   - 推演结果下方会生成标准AI指令
   - 包含您的问题和三个用神

2. **一键复制**
   - 点击"📋 复制指令"按钮
   - 按钮会显示"✓ 已复制到剪贴板"
   - 手机会轻微震动（支持设备）

3. **使用指令**
   - 打开ChatGPT、Claude或其他AI工具
   - 粘贴复制的指令

## 🔧 常见问题解决

### 安装问题

#### Q: 手机上找不到"添加到主屏幕"按钮？
**A: 解决方案**
- Android：确保使用Chrome浏览器最新版本
- iPhone：确保使用Safari浏览器（iOS 13+）
- 清除浏览器缓存后重试

#### Q: 安装后图标不显示或显示异常？
**A: 解决方案**
- 重启手机
- 重新安装应用
- 清除浏览器数据

### 使用问题

#### Q: 数字输入无响应？
**A: 解决方案**
- 检查输入框是否聚焦
- 确认输入的是1-999之间的整数
- 尝试点击其他位置后再试

#### Q: 阴历日期选择有问题？
**A: 解决方案**
- 确保按顺序选择：月份→日期→时辰
- 检查所有选项是否都已选择
- 重新刷新页面

#### Q: 复制按钮不工作？
**A: 解决方案**
- 确保浏览器支持剪贴板功能
- HTTPS网站需要用户授权
- 可以手动长按选择复制

### 显示问题

#### Q: 文字显示不清楚或乱码？
**A: 解决方案**
- 刷新页面重试
- 检查系统字体支持中文
- 调整浏览器字体大小

#### Q: 界面布局错乱？
**A: 解决方案**
- 确保浏览器是最新版本
- 清除浏览器缓存
- 尝试其他浏览器
